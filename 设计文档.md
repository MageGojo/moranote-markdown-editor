# Markdown 编辑器 - 设计文档

## 1. 项目概述

基于 GPUI 框架开发的桌面端 Markdown 编辑器，支持实时预览、文件管理和基本编辑功能。

### 1.1 项目目标

- 提供流畅的 Markdown 编辑体验
- 支持文件夹管理和文件操作
- 实现实时渲染预览
- 提供可配置的用户设置

## 2. 技术栈

| 组件 | 技术选型 |
|------|----------|
| GUI 框架 | GPUI (Zed 编辑器框架) |
| 异步运行时 | Tokio |
| 组件库 | GPUI 组件库 |
| 预览渲染 | Webview |

## 3. 功能需求

### 3.1 核心功能

1. **文件夹管理**
   - 打开文件夹
   - 浏览文件树
   - 文件夹展开/折叠

2. **文件操作**
   - 删除文件
   - 重命名文件
   - 新建文件

3. **编辑功能**
   - Markdown 语法高亮
   - 实时编辑
   - 快捷键支持

4. **预览功能**
   - Webview 实时预览
   - 编辑/预览同步滚动
   - 支持标准 Markdown 语法渲染

5. **设置界面**
   - 主题切换
   - 字体大小调整
   - 编辑器配置

## 7. 数据流设计

### 7.1 编辑预览同步流程

```
用户输入 → 编辑器捕获 → Markdown 解析 → HTML 转换 → Webview 渲染
    │                                                    │
    └──────────────────── 同步滚动 ←─────────────────────┘
```

### 7.2 文件操作流程

```
用户操作 → 文件管理器 → 文件系统操作 → 更新 UI
    │              │
    └──────────────┴→ 错误处理 → 提示用户
```
